A Guide to CSS Selectors + Combinators

CSS or Cascading Style Sheets, is a stylesheet language used to provide styling and aesthetic control to a webpage that uses HTML. CSS is useful in that it allows one to control how massive amounts of information are presented on a webpage.

This website will provide information on how the primary jargon of CSS communicates to a browser. This is done through the use of symbols known as selectors and combinators.
