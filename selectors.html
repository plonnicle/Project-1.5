CSS Selectors

CSS selectors are symbols that define how styling rules are applied to a webpage. Selectors act as operators that correspond to specific tags in the HTML code. This connection allows for facile and definitive formatting and coloring. These selectors include:

# (id)

 Example 1 -

HTML:
<tag id = "example"></tag>
 CSS:
#example { }

. (class)

 Example 2 -

HTML:
<tag class = "example"></tag>
 CSS:
.example { }

/nothing/ (element)

 Example 3 -

HTML:
<p></p>
CSS:
p { }
