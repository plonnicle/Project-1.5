CSS Combinators Pt. 1

In CSS, combinators are another form of communicating with HTML about which elements and segments of code have particular styling applied to them. The difference is between combinators and selectors is that combinators depict relations between different blocks of code. This section of the website will describe the parent-child and ancestor-descendant combinators.

>

For example, the > symbol designates the relationship between a parent and child block of code:

Example 1 -

HTML:
<parent>
<child></child>
</parent>
CSS:
parent>child

________________________

[space]

Similarly, the [space] symbol designates a similar relationship between a greater block of code and a lesser. This [space] symbol however more generally applies to any block of code within a greater initial block; not necessarily the code block directly underneath it.

This relationship is known as the ancestor-descendant:

Example 2 -

HTML:

<ancestor>{
...{
...{
<descendant>
}
}
}

CSS:

ancestor descendant
